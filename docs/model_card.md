# DCLC Model Card

- **Language**: Sinhala (si).
- **Model type**: Decoder-only transformer, 500M parameters (24 layers, 768 hidden, 12 heads).
- **Capabilities**: Text generation, image generation (latent VQ), image editing.
- **Content policy**: No built-in content restriction. Explicit content is allowed by default. Optional filter can be enabled by the user. See [USAGE_POLICY.md](../USAGE_POLICY.md).
- **Training**: Sinhala corpus; optional image captions; optional explicit data path (gitignored).
