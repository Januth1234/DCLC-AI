# Installation

- **Python**: 3.10 or newer.
- **Windows**: Recommended. Use PowerShell or Command Prompt.

## Steps

1. Clone the repo and `cd DCLC-AI`.
2. Create a virtual environment: `python -m venv .venv` then `.venv\Scripts\activate`.
3. Install: `pip install -r requirements.txt`.
4. Optional (build exe): `pip install -r requirements-app.txt`.
5. Run: `python -m app.main`.

## Windows 10+

DCLC supports Windows 10 and later. If antivirus blocks the app or PyInstaller build, add an exclusion for the project folder.
